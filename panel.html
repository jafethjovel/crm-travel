Configuración de Entorno CRM

Conecta tu PC principal con tu laptop para desarrollo

Esta guía te ayudará a configurar tu PC como servidor principal y tu laptop como cliente

1

Configuración de Red

Encontrar la IP de tu PC principal

Abre la línea de comandos (CMD) en tu PC y ejecuta:

ipconfig

Busca la "Dirección IPv4" y anótala. Será algo como 192.168.1.XX

Configurar IP estática (recomendado)

Para evitar que la IP cambie, configura una IP estática en tu PC:

// En Windows:
1. Ve a Centro de redes y recursos compartidos
2. Haz clic en "Cambiar configuración del adaptador"
3. Haz clic derecho en tu adaptador de red → Propiedades
4. Selecciona "Protocolo de Internet versión 4 (TCP/IPv4)" → Propiedades
5. Selecciona "Usar la siguiente dirección IP"
6. Ingresa una IP fija (ej: 192.168.1.100)
7. Máscara de subred: 255.255.255.0
8. Puerta de enlace predeterminada: 192.168.1.1 (usa la de tu router)

Configuración de XAMPP

Permitir conexiones externas en MySQL

En tu PC principal, abre el archivo de configuración de MySQL:

1. Ve a la carpeta de XAMPP (normalmente C:\xampp)
2. Abre mysql\bin\my.ini
3. Busca la línea [mysqld]
4. Agrega o modifica: bind-address = 0.0.0.0
5. Guarda el archivo y reinicia MySQL desde el panel de control de XAMPP

Crear usuario para acceso remoto

Abre PHPMyAdmin en tu PC principal y ejecuta:

-- Crear un usuario para acceso remoto
CREATE USER 'usuario_remoto'@'%' IDENTIFIED BY 'password_seguro';

-- Otorgar todos los privilegios (ajusta según necesites)
GRANT ALL PRIVILEGES ON *.* TO 'usuario_remoto'@'%' WITH GRANT OPTION;

-- Aplicar los cambios
FLUSH PRIVILEGES;

Configurar el firewall de Windows

Permite el tráfico en los puertos necesarios:

1. Abre el Firewall de Windows
2. Ve a "Configuración avanzada"
3. Haz clic en "Reglas de entrada" → "Nueva regla"
4. Selecciona "Puerto" → Siguiente
5. Puerto TCP: 80 (HTTP), 3306 (MySQL)
6. Permitir la conexión → Siguiente
7. Marca todos los perfiles → Siguiente
8. Asigna un nombre (ej: "XAMPP Remote") → Finalizar

Configuración en la Laptop

Conectar desde la laptop al PC principal

En tu laptop, configura tu proyecto para usar la IP del PC principal:

// En tu archivo de conexión a la base de datos (PHP)
$host = "192.168.1.100"; // IP de tu PC principal
$user = "usuario_remoto";
$password = "password_seguro";
$database = "nombre_base_datos";

$conn = new mysqli($host, $user, $password, $database);

Probar la conexión

Desde tu laptop, prueba si puedes acceder al servidor:

// Abre el navegador en tu laptop y visita:
http://[IP-DE-TU-PC]/tu_proyecto

// Para probar MySQL:
// Crea un archivo test_connection.php con el código de conexión
